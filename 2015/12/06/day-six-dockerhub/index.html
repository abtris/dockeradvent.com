Day Six - Docker Hub

06 Dec 2015

Hi everyone,
Docker Hub is place to store container images. If you create new image, you need push them into registry. Registry can be public (default is Docker Hub) or private.

Registry you can have as service:

  • Docker Hub
  • Docker Trusted Registry
  • Quay
  • Tutum

or you can run them alone:

  • Docker Registry 2.0

It's easy to run registry myself using docker too. I used docker registry with S3 and Nginx for authentication.
