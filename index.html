Day One - Open Container Ekosystem

02 Dec 2015

Hi everyone,
I can talk about Open Container Ecosystem, today.

If you look at mindmap created by Krishnan Subramanian. You can see how is open container ecosystem wide.

You have in mindmap big categories: - Container OS - Application Deployment/Development platforms - Developer tools - Hosting/Service providers - Database/Big data - Config Management Tools with Docker support - Networking - Image registry - Monitoring - Security - Scheduler/Orchestration/Management/Monitoring - Misc

Some of them are cool and I make some posts about them, some can be split into more categories.

Tomorrow we introduce how install docker and start our first container.

Regards, Ladislav

Hi everyone,
I see docker as two things. First is Docker platform created by Docker, Inc that includes Docker client/server, Dockerhub and docker tools (machine, swarm, compose). I make post about every tool.

Second is The Open Container Initiative about definition what is container and runc as first implementation. Docker made this with others companies and this is big.

Open Container ecosystem is huge and you see them in next post.
